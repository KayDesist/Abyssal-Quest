# Abyssal Quest

A browser-based turn-based RPG built with C# and ASP.NET Core. Still a work in progress.

## What it is
You pick a character class and fight your way through ocean depth zones in turn-based combat. Each zone has a boss. Go as deep as you can.

## Stack
- C# / ASP.NET Core MVC
- Razor Views
- .NET 8.0

## Status
Early development. Project structure is set up, implementation coming soon.
